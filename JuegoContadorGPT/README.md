# JuegoContador

App desarrollada en React para un desafío técnico.

## Cómo ejecutar

1. Instalar dependencias:
```bash
npm install
```

2. Ejecutar:
```bash
npm start
```

## Funcionalidad

- Botón de inicio
- Cuenta regresiva: Preparados, Listos, Ya
- Juego de 5 segundos
- Contador de clicks
- Puntaje máximo

## Supuestos

- Se usa React sin librerías externas para simplificar
- Diseño simple enfocado en funcionalidad
